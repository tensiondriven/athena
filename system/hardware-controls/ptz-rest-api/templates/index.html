API Information

This system provides REST API endpoints for PTZ camera control:

  • GET /api/cameras - List cameras
  • POST /api/cameras/{id}/pan-tilt - Control pan/tilt
  • POST /api/cameras/{id}/zoom - Control zoom
  • POST /api/cameras/{id}/preset - Set/recall preset
  • POST /api/cameras/{id}/screenshot - Capture screenshot
